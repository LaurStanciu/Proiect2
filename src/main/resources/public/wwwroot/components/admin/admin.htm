Storage Configuration

In this page you can set up your connection with the database in two ways:

  • complete the form below and press the "Save" button
  • or press the "Use default settings" button to use the default settings

Select the database provider.
Enter the server database name. Server Name must be between 2 and 50 characters long!
Enter the database name Database Name must be between 3 and 50 characters long!
Enter the user id to login to database. User Name must be between 1 and 50 characters long!
Enter the password to login to database. Password must be between 5 and 50 characters long!
